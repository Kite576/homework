# 在图片上叠加图片并居中显示 我将创建一个HTML页面，展示如何在一个图片上叠加另一个图片，并使第二张图片居中显示在第一张图片上。 ## 设计思路 1. 使用容器包裹两张图片 2. 第一张图片作为背景图 3. 第二张图片使用绝对定位并居中 4. 添加交互效果展示不同实现方式 ## 实现代码 ```html 图片叠加居中效果

图片叠加居中效果

使用CSS实现在一张图片上叠加另一张图片，并使叠加的图片居中显示

绝对定位方法

这是最常用的方法：

  1. 将容器设置为相对定位（position: relative）
  2. 背景图片正常显示
  3. 叠加图片设置为绝对定位（position: absolute）
  4. 使用 top: 50% 和 left: 50% 定位到容器中心
  5. 使用 transform: translate(-50%, -50%) 微调位置使图片真正居中
<div class="image-container"> <img src="background.jpg" class="background-img"> <img src="overlay.png" class="overlay-img"> </div> .image-container { position: relative; } .background-img { display: block; width: 100%; } .overlay-img { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); max-width: 40%; }

CSS Grid方法

使用CSS Grid布局实现：

  1. 将容器设置为网格布局（display: grid）
  2. 使用单个单元格（grid-template-areas: "overlay"）
  3. 两张图片都放在同一个网格区域
  4. 背景图片铺满整个容器
  5. 叠加图片使用 align-self 和 justify-self 居中
<div class="image-container"> <img src="background.jpg" class="background-img"> <img src="overlay.png" class="overlay-img"> </div> .image-container { display: grid; grid-template-areas: "overlay"; } .background-img, .overlay-img { grid-area: overlay; } .background-img { width: 100%; height: 100%; object-fit: cover; } .overlay-img { align-self: center; justify-self: center; max-width: 40%; }

Flexbox方法

使用Flexbox布局实现：

  1. 将容器设置为Flex布局（display: flex）
  2. 背景图片绝对定位并铺满容器
  3. 叠加图片作为Flex项目
  4. 使用margin: auto使叠加图片居中
  5. 需要设置容器高度
<div class="image-container"> <img src="background.jpg" class="background-img"> <img src="overlay.png" class="overlay-img"> </div> .image-container { display: flex; position: relative; height: 400px; /* 需要设置高度 */ } .background-img { position: absolute; width: 100%; height: 100%; object-fit: cover; } .overlay-img { margin: auto; max-width: 40%; position: relative; /* 确保图片显示在顶层 */ }
